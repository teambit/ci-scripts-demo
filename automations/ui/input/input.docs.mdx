---
description: A reusable Input component with support for labels, helper text, error states, and icons.
---

import { Input } from './input';

## Basic usage

Use the input component with a label and placeholder:

```js
<Input 
  label="Username"
  placeholder="Enter your username"
/>
```

### With helper text

```js live
<Input 
  label="Email"
  placeholder="Enter your email"
  helperText="We'll never share your email"
/>
```

### With error state

```js live
<Input 
  label="Password"
  type="password"
  placeholder="Enter your password"
  error="Password is required"
/>
```

### With icons

```js live
<Input 
  label="Search"
  placeholder="Search..."
  icon="🔍"
  iconAfter="✓"
/>
```
