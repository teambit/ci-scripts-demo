# Header Component

A flexible and responsive header component that includes a logo area, navigation menu, and user menu.

## Usage

```jsx
import { Header } from '@your-scope/design.patterns.header';

const navItems = [
  { label: 'Home', href: '/' },
  { label: 'About', href: '/about' },
  { label: 'Services', href: '/services' },
];

const userMenuItems = [
  { label: 'Profile', onClick: () => console.log('Profile clicked') },
  { label: 'Settings', onClick: () => console.log('Settings clicked') },
];

<Header
  logo={<YourLogo />}
  navItems={navItems}
  userMenuItems={userMenuItems}
  fixed={true}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| logo | ReactNode | undefined | The logo to display in the header |
| navItems | Array<{label: string, href: string}> | [] | Navigation items to display |
| userMenuItems | Array<{label: string, onClick: () => void}> | [] | User menu items to display |
| fixed | boolean | false | Whether the header is fixed at the top |
